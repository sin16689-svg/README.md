# 리눅스 명령어

## 1.top
- 시스템의 실시간 성능 상태와 프로세스 목록을 보여주는 명령어

---

### System Summary(상단 부분):
시스템에 대한 전체적인 상태 요약 정보

- 현재 시간, 시스템 부팅 시간 및 업타임
- CPU, 메모리, 스왑 사용량(사용량, 가용량, 전체 용량)
- 프로세스 총 수, 실행 중인 프로세스 수, 슬립 중인 프로세스수
- CPU 사용량(user, system, idel)
  
### Task/Process List(하단 부분):

- 각 프로세스에 대한 상세 정보(PID, 사용자, CPU사용량, 메모리 사용량 등)

---

### 주요 키 설명

| 주요 키 | 기능 설명 |
|----|-----------|
| **q** | top 종료 |
| **h** | 도움말 표시(사용 가능한 명령어 목록 출력) |
| **P** | CPU 사용률 기준 정렬 |
| **M** | 메모리 사용량 기준 정렬 |
| **T** | 실행 시간 기준 정렬 |
| **k** | 프로세스 종료(입력한 PID를 종료) |
| **r** | 프로세스 우선순위 변경(PID + 새로운 우선순위 입력) |
| **z** | 컬러 모드 전환 |
| **c** | 명령어 라인 표시/숨기기(프로세스 실행 커맨드 표시) |
| **1** | CPU 코어별 사용량 표시/숨기기 |
| **u** | 특정 사용자 프로세스만 보기 |
| **i** | idle 프로세스 숨기기 |
| **d / s** | 갱신 주기(초) 변경 |
| **Shift + R** | 정렬 순서 뒤집기 |
| **f** | 컬럼 선택 메뉴 |
| **o** | 정렬 기준 변경 |

---

### 명령어 옵션 

| 옵션 | 설명 |
|------|--------------------------|
| `-d 초` | 새로고침 주기 변경 |
| `-u 사용자` | 특정 사용자 프로세스만 표시 |
| `-p PID` | 특정 프로세스만 모니터링 |
| `-n 횟수` | 지정된 횟수만 갱신 후 종료 |

---

### 활용 예시 

```bash
top               # 기본 실행
top -d 1          # 1초마다 갱신
top -u student    # 특정 사용자 프로세스 보기
top -p 1234       # 특정 PID 모니터링

---

### 활용 상황

- 서버 CPU 사용률이 갑자기 증가했을 때 원인 프로세스 확인
- 메모리 누수 의심 프로세스 추적
- 특정 프로그램의 리소스 사용 추세 모니터링
- 백그라운드에서 리소스 과사용 프로세스 탐지
