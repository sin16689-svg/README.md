# README.md

1. top

top 명령어는 시스템의 실시간 프로세스 상태 및 리소스 사용량을 모니터링할 수 있는 도구입니다.

✔ 주요 기능

CPU, 메모리 사용량 실시간 확인

실행 중인 프로세스 확인

특정 기준으로 정렬하여 모니터링 가능

✔ 사용 예시
top              # 기본 실행
 top -u 사용자명   # 특정 사용자 프로세스만 보기
top -p PID       # 특정 PID 프로세스만 보기
✔ top 화면 주요 항목 설명

PID : 프로세스 ID

USER : 프로세스를 실행한 사용자

%CPU : CPU 사용률

%MEM : 메모리 사용률

TIME+ : CPU가 사용된 총 시간
